Key Features of the New Modal Design:

  • ✅ Modal Overlay: Opens as a popup instead of navigating to a new page
  • ✅ Sidebar Navigation: Left sidebar with organized sections (Overview, Legal Details, Proceedings, etc.)
  • ✅ Professional Layout: Clean, lawyer-friendly design with consistent styling
  • ✅ Better UX: Users stay on the search page and can quickly view case details
  • ✅ Responsive Design: Works well on different screen sizes
  • ✅ Keyboard Support: ESC key closes the modal
  • ✅ Click Outside to Close: Click outside the modal to close it

Navigation Sections:

  • Overview: Basic case information (case number, court, status, dates)
  • Legal Details: Case stage, orders, disposal information, advocates
  • Proceedings: Civil Miscellaneous Applications table
  • Court Orders: Detailed orders with document access
  • Documents: Document management and downloads
  • FIR Information: Criminal case details (when applicable)
